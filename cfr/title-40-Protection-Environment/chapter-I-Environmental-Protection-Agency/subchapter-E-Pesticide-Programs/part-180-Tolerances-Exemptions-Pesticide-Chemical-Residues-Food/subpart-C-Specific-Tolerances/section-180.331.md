##### ยง 180.331 4-(2,4-Dichlorophenoxy) butyric acid; tolerances for residues. #####

(a) *General.* Tolerances are established for residues of the herbicide 4-(2,4-dichlorophenoxy) butyric acid (2,4-DB), both free and conjugated, determined as the acid, in or on food commodities, as follows:

|       Commodity       |Parts per million|
|-----------------------|-----------------|
|    Alfalfa, forage    |       0.7       |
|     Alfalfa, hay      |       2.0       |
|Cattle, meat byproducts|      0.05       |
|    Clover, forage     |       0.2       |
|      Clover, hay      |       0.2       |
| Goat, meat byproducts |      0.05       |
| Hog, meat byproducts  |      0.05       |
|Horse, meat byproducts |      0.05       |
|        Peanut         |       0.2       |
|   Peppermint, tops    |       0.2       |
|Sheep, meat byproducts |      0.05       |
|    Soybean, forage    |       0.7       |
|     Soybean, hay      |       2.0       |
|     Soybean, seed     |       0.5       |
|    Spearmint, tops    |       0.2       |
|    Trefoil, forage    |       0.7       |
|     Trefoil, hay      |       2.0       |
