##### ยง 180.284 Zinc phosphide; tolerances for residues. #####

(a) *General.* Tolerances are established for residues of the phosphine resulting from the use of the rodenticide zinc phosphide in or on the raw agricultural commodities as follows:

|       Commodity        |Parts per million|
|------------------------|-----------------|
|    Alfalfa, forage     |       0.2       |
|      Alfalfa, hay      |       0.2       |
|     Barley, grain      |      0.05       |
|      Barley, hay       |       0.2       |
|     Barley, straw      |       0.2       |
|    Bean, dry, seed     |      0.05       |
|   Beet, sugar, roots   |      0.05       |
|   Beet, sugar, tops    |       0.2       |
|         Grape          |      0.01       |
|Grass, rangeland, forage|       0.1       |
| Grass, rangeland, hay  |       0.1       |
|         Potato         |      0.05       |
|    Sugarcane, cane     |      0.01       |
|      Timothy, hay      |       0.5       |
|    Timothy, forage     |       0.5       |
|     Wheat, forage      |      0.05       |
|      Wheat, grain      |      0.05       |
|       Wheat, hay       |      0.05       |
|      Wheat, straw      |      0.05       |
