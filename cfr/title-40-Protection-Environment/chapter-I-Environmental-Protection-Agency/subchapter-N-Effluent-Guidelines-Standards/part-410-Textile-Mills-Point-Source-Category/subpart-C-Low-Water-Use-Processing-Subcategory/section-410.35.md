|                                                      |    Kg/kkg (pounds per 1,000 lb) of product    |              |
|                         BOD5                         |                      1.4                      |     0.7      |
|                         COD                          |                      2.8                      |     1.4      |
|                         TSS                          |                      1.4                      |     0.7      |
|                          pH                          |                (<sup>1</sup>)                 |(<sup>1</sup>)|
|<sup>1</sup> Within the range 6.0 to 9.0 at all times.|                                               |              |
