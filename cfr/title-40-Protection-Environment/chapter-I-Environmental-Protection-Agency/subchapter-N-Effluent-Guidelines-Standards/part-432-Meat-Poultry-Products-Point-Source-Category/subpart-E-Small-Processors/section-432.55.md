##### ยง 432.55 New source performance standards (NSPS). #####

Any source that is a new source subject to this subpart must achieve the following performance standards:

|                    Regulated  <br/>parameter                     |Maximum daily <sup>1</sup>|Maximum monthly avg. <sup>1</sup>|
|------------------------------------------------------------------|--------------------------|---------------------------------|
|                         BOD<sub>5</sub>                          |           1.0            |               0.5               |
|                          Fecal Coliform                          |      (<sup>2</sup>)      |         (<sup>2</sup>)          |
|                         O&G <sup>3</sup>                         |           0.5            |              0.25               |
|                               TSS                                |           1.2            |               0.6               |
| <sup>1</sup> Pounds per 1000 lbs (or g/kg) of finished product.  |                          |                                 |
|                   <sup>2</sup> No limitation.                    |                          |                                 |
|<sup>3</sup> May be measured as hexane extractable material (HEM).|                          |                                 |
